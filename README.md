# File Organizer 📁

A script to organize files into folders based on their extensions.

## 📋 Features
- Automatically creates folders for extensions.
- Moves files into corresponding folders.

## 🚀 Quick Start
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/file-organizer.git
2. Place files in the files_directory folder.
3. Run the script:
python file_sorter.py

Example Usage:
Before:
files_directory/
  file1.txt
  file2.jpg

After:
files_directory/
  txt/
    file1.txt
  jpg/
    file2.jpg
